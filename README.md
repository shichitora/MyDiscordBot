# 七虎なるくん
荒らし対策に特化した多機能ボットです

このコードを参考に使用した場合は「七虎なる」と以下のルール及びボットの利用規約に従ってください
1. コードの完全コピーは禁止です
2. このコードを使用して悪用することは禁止です
3. 自分のものだと偽ることは禁止です
4. 使用した場合はこのレポジトリのURLを貼る必要があります
5. 七虎なるが不適切やこれはダメだろと判断した場合は即ソースコード非公開にして、そのユーザーに最適な処置をします。

また機能のコピー対策として、一部のコードまたは過去に使用したコードのみ公開しています。
